About This Project

This is an IoT oven temperature monitoring application that displays real-time temperature readings from an AWS IoT-connected device.

How It Works

  • Temperature sensor in the oven sends data to AWS IoT Core via MQTT
  • Server receives MQTT messages and broadcasts to connected clients via WebSocket
  • Client displays live temperature data with a 3D oven visualization

Technologies Used

  • Three.js for 3D graphics
  • WebSocket for real-time communication
  • AWS IoT Core for device connectivity
